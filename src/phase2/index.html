🎵 YM2151 WebAudio Player

Phase 2: Nuked-OPM WASM + AudioWorklet Implementation

このプログラムは、Nuked-OPMエミュレーター（WASM）とAudioWorkletを使用して、440Hzの音声を3秒間演奏します。

FM音源レジスタのwrite後には、10msのcycle消費を行います。

実際のYM2151エミュレーションによるリアルタイム音声合成を実装しています。

周波数: 440Hz (A4)
演奏時間: 3秒
プラットフォーム: WASM + AudioWorklet
レジスタ遅延: 10ms/write
